# Foldername Virus Killer
## What can this do
This is a simple batch file created to kill the specific type of annoying
autorun virus that always hides all folders in a directory and create a useless
executable file with the same name as the original folder.
## How to use
Download the batch file to the infected directory and run
[link](https://raw.githubusercontent.com/WangNianyi2001/Foldername-Virus-Killer/master/killer.bat)
